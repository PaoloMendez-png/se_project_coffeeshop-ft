# Triple Peaks Coffee Shop

This is the second project of the fulltime Software Engineering program at TripleTen. It was created using HTML and CSS, based on the design brief.

## Project features

- Semantic HTML5
- Flexbox
- Positioning
- Flat BEM file structure
- A custom form
- CSS animation and transform

## Plan on improving the project

In regards to improvement although the page already looks amazing personally what I do want to get involved with is adding a small button around the right, bottom right part of the screen with a simple "UP" arrow that stays on screen that way when the user engages with the nav, instead of having to scroll all the way to the top they can just click the button to the right to immediately bring them back.

(This might not be much of an effect on phones since they can quickly reach the top just by tapping the top bit of the screen but would be a nice addition for users operating on a computer.)
